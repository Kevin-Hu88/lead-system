# 电话话术卡 - 快速筛选有需求的客户

## 开场（10秒）

> 您好，请问是XX（公司名/店名）吗？我们是武汉本地做膜结构车棚、电动遮阳棚的厂家，想问下贵单位停车场现在有遮阳雨棚吗？

## 快速筛选（30秒判断）

### 情况1：已经有棚了
> 好的，那打扰了。如果以后需要翻新或者加装，随时联系我们，祝您生意兴隆！
→ 标记"已有棚"，不跟进

### 情况2：没有棚，但没需求
> 好的，我们免费上门测量出图，如果以后有需要可以加个微信，我发些案例给您参考。
→ 加微信，标记"暂无需求"，3个月后回访

### 情况3：没有棚，有兴趣
> 太好了！我们是武汉本地厂家，质保10年，可以免费上门测量出方案。方便加个微信吗？我把案例和报价发给您看看。
→ **重点跟进！** 标记"有意向"，当天发案例+约上门

### 情况4：正在找/正在比价
> 您在找车棚厂家是吧？我们做了很多本地项目，XX小区、XX工厂都是我们做的。方便加微信发案例吗？价格方面厂家直供，肯定比外面实惠。
→ **高意向！** 标记"热线索"，当天跟进报价

## 针对不同客户类型的话术调整

### 4S店 / 汽车店
> 您好，我们做膜结构车棚的，很多4S店展厅外都装了我们的棚，既能遮阳挡雨，又能提升店面形象。您这边室外停车区有棚吗？

### 工厂 / 工业园
> 您好，我们做钢结构车棚的，厂区停车棚、员工电动车棚都能做。您这边厂区有停车棚的需求吗？

### 充电站 / 充电桩
> 您好，我们做充电桩配套车棚的，光伏车棚既能遮阳又能发电。您这边充电站有装车棚的计划吗？

### 学校 / 医院
> 您好，我们做非机动车停车棚的，很多学校和医院都装了我们的棚，家长等候区、病人都能遮阳挡雨。

### 招标项目（S级）
> 您好，看到贵单位的XX车棚招标公告，我们是武汉本地膜结构厂家，有相关资质和施工经验，想了解下项目详细情况。

## 挂电话后必做

1. 在"触达跟进表"记录：日期、客户反应、下一步行动
2. 在系统里更新线索状态：有意向/已联系/无需求
3. 有意向的当天发微信案例图片

## 每日目标

- 新线索筛选：30-50个电话/天
- 有效通话（聊了30秒以上）：15-25个
- 有意向客户：2-5个/天
- 约上门测量：1-2个/天
